Необходимо реализовать простой IM-клиент: 

на одной HTML-странице
расположить два фрейма, в каждом из которых есть следующие элементы:
         Поле ввода псевдонима контакта. Все сообщения в чате должны
начинаться с этого имени отправителя.
         Поле чата, в котором показываются все сообщения, отправленные и
полученные этим контактом.
         Поле ввода сообщения.
         Кнопка отправки сообщения.

При отправке сообщения из одного фрейма оно должно появляться в другом
фрейме. Отправка должна реализовываться посредством HTML5 Post Message
либо любой обертки над ним. Все сообщения должны сохраняться в HTML5
Session Storage (можно использовать обертки) корневого фрейма страницы
(чтобы можно было смотреть в Inspect'е). Персистентность сообщений не
требуется (при перезагрузке страницы сообщения пропадают).

Дополнительные требования:
         Если в тексте сообщения встречается имя одного из контактов, то
оно должно быть выделено (болд, цвет, андерлайн или что-то еще).
         Если в тексте сообщения встречается код смайлика, он должен
заменяться на картинку (можно реализовать для 4-5 основных смайлов).

По верстке требований нет, можно не заморачиваться, интересует функционал.
Server-side не требуется, все может крутиться локально в рамках HTML-страницы.
Если есть какие-то
вопросы - задавайте, уточним.

Если получится то используйте react или angular и какой то сборщик для локального запуска.
